# Job Market Analysis

A comprehensive analysis of 16,000+ job postings using Python and data visualization.

## 📊 Overview

This project analyzes job market trends to uncover:
- Most in-demand job titles
- Geographic job market hotspots  
- Salary ranges and trends
- Experience level distribution
- Employment type breakdown

## 📁 Project Structure
job-market-analysis/
├── data/
│   ├── raw/              # Original job postings CSV
│   └── processed/        # Cleaned dataset
├── notebooks/            # Jupyter analysis notebooks

│   ├── 02_data_cleaning.ipynb

## 🔍 Key Findings

### Top Job Titles
- Sales Director (Owner/Operator) - 83 positions
- Sales Director variations - 78 positions
- Sales Associate - 65 positions
- Retail Sales Associate - 60 positions
- Staff Accountant - 42 positions

### Top Markets
- United States (general) - 1,133 jobs
- New York, NY - 398 jobs
- Chicago, IL - 267 jobs
- Houston, TX - 243 jobs
- Atlanta, GA - 207 jobs

### Employment Types
- Full-time: 12,844 jobs (88%)
- Contract: 1,739 jobs (12%)
- Part-time: 1,010 jobs

### Experience Levels
- Mid-Senior level: 5,883 jobs (40%)
- Entry level: 3,694 jobs (25%)
- Associate: 1,220 jobs (8%)

### Salary Insights
- Average: $41,168
- Median: $26
- Range: $10 - $998,426

## 🛠️ Technologies Used

- **Python 3** - Programming language
- **Pandas** - Data manipulation & analysis
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter Notebook** - Interactive analysis

## 📈 Visualizations

All charts are saved as high-resolution PNG files (300 DPI):
- Top 10 job titles (bar chart)
- Top 15 locations (bar chart)
- Work type distribution (bar chart)
- Experience level distribution (pie chart)

## 💡 Key Insights

✅ Sales roles dominate the dataset with Sales Directors as the most common position

✅ Diverse geographic distribution across the United States

✅ Full-time employment is the primary work type (88% of jobs)

✅ Mid-senior level positions are most common

✅ Wide salary range indicates significant variation by role and location
